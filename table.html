CUA Alternate Menu item Function
Ctrl + Del Ctrl + Del Edit | Clear Removes selected text from window; doesn't put it in Clipboard
Ctrl + Ins Ctrl + Ins Edit | Copy Copies selected text to Clipboard
Shift + Del Shift + Del Edit | Cut Places selected text in Clipboard, deletes selection
Shift + Ins Shift + Ins Edit | Paste Pastes text from Clipboard into the active window
